# 📝 Todo App

A modern Todo application built with ReactJS that helps users manage daily tasks efficiently. The app supports full CRUD operations, task searching, filtering, drag-and-drop reordering, and local storage persistence.

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* 🔍 Search tasks instantly
* ✅ Mark tasks as completed or pending
* 📂 Filter tasks (All / Pending / Completed)
* ↕️ Drag and drop to reorder tasks
* 💾 Local Storage support (tasks persist after page refresh)
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Local Storage API
